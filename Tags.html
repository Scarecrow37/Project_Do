  • !doctype _ (Document Type): Custom of coding.
  • html(HyperText Markup Language) : open and close the coding.
  • head(Head) : Container for all the head elements.
  • body(Body) : Container for all the body elements.
  • meta(Meta) : Provides metadata about the HTML document.
  • title(Title) : Title of web site
  • h1~h6(Heading 1~6) : Define HTML headings.
  • ol(Ordered List) : Ordered list item(li) to number.
  • ul(Unordered List) : Ordered list item(li) to dot.
  • li(Line Item) : Defines a list item.
  • p(Paragraph) : Defines a paragraph.
  • br(Line Breaker) : Inserts a single line break.
  • strong(Strong) : Bold text.
  • u(Underline) : Underline text.
  • a(Anchor) : Defines a hyperlink.
  • img(Image) : Inserts a image.
  • table(Table) : Inserts a table.
  • tr(Table Row) : Line of table
  • td(Table Cell) : Cell of table
  • input(Input) : Insert someting
